# ALTEN SHOP FRONT

Front:

pour se connecter au front:
	=> taper npm install to download node_modules
	=> npm run start to start the application
	=> pour se connecter comme admin use admin@admin.com

Back:

Pour lancer le back :
	=>import maven project
	puis run application
les api sont testable via postman
	=> pour tester importer la collection Altenshop Collection.postman_collection
	=> aussi importer l'environement (baseurl and token setting) suivant altenshop env.postman_environment
	=> utiliser cet environnement comme env par defaut (se situe top right)
